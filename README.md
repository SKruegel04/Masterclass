# Masterclass
Mit Masterclass können Kurse angelegt und verwaltet werden.
Kursgeber können sich anmelden und Kurse anlegen und terminieren sowie Teilnehmerlisten einsehen.
User können Kurse buchen und wieder abzusagen.
